TabMate privacy

Privacy policy for TabMate.

TabMate is a browser extension that opens your chosen assistant from any Chrome tab and can send highlighted text with page context into that assistant.

Data handling

What TabMate touches.

1

Selected text and page context.

When you use Explain, Ask, or Simplify, TabMate reads the text you selected plus the current page title and URL so it can send that context into ChatGPT, Claude, or Gemini.

2

Stored preference only.

TabMate stores your selected default assistant in Chrome extension storage so the corner button knows which provider to open.

3

No remote TabMate server.

TabMate does not run its own backend for chats. Your prompt is handed off directly to the provider you chose through the provider website already open in Chrome.
